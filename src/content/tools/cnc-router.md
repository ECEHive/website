The Hive's machine shop is located on the first floor and provides resources for students to design and build the structural components of their projects. A PI must be present in the room for you to use any machine shop tools.

## Power Tools (SUMS Access Required)

- **Miter saw**
- **Drill presses**
- **Scroll saw**
- **Band saw**
- **Belt sanders**
- **Sandblasting booth**

## Hand Tools

- Cordless drills (with drill bits and screws)
- Dremel rotary tools
- Hand saws, jigsaw, reciprocating saw
- Sandpaper

## Materials

Scrap wood is available in a large red bin inside the machine shop. For specific materials, check the local hardware store or order online.

## Training

A PI must be present in the room at all times when the machine shop is in use. Reading SOPs alone is not sufficient for training.
